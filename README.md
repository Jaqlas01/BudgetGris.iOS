# BudgetGris.iOS
Budget App 

BudgetGris

BudgetGris er en nem app, der giver overblik over dit budget – uden login og uden indsamling af data.

⸻

Privatlivspolitik

BudgetGris indsamler, gemmer eller deler ingen personlige data.

Alle data gemmes udelukkende lokalt på din enhed og bliver ikke sendt til nogen server eller tredjepart. Appen anvender ikke login, tracking eller reklamer.
